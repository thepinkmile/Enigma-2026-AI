# Enigma Z (Zahlenmaschine / Numbers Enigma)

| Field | Value |
| ------- | ------- |
| Who | ChiMaAG (Chiffriermaschinen AG), Berlin |
| What | Numbers-only Enigma variant with 10-contact rotors (digits 0–9); designed for encrypting numerical messages (weather reports, pre-coded text) |
| When | ~1930; offered to Spain November 1931 |
| Where | Manufactured in Berlin, Germany (52.5200°N, 13.4050°E); sold to Sweden (STAB/Ivar Kreuger) and Chile |
| Related | [Arthur Scherbius](../profiles/arthur-scherbius.md), [Enigma D Commercial](enigma-d-commercial.md) |

## Overview

The Enigma Z (Z30, internal Ch.16) is the only numbers-only Enigma machine. Where all other Enigma variants operate on a 26-letter alphabet, the Enigma Z has only 10 contacts per rotor — one for each
digit 0–9. It was designed for encrypting purely numerical messages such as weather data and pre-coded numeric traffic. The machine exists in two versions: Mark I (regular stepping) and Mark II
(cogwheel stepping).

## Versions

### Z30 Mark I (S/N Z101–Z150)

- Regular Enigma stepping (double-stepping anomaly present)
- Driven UKW (10 positions)
- Period: 9,910
- Power: hell/dunkel/aus (3 positions)
- Sold to Sweden (Z101–Z103); OKW/Chi (Z106)

### Z30 Mark II (S/N Z151–Z200)

- Cogwheel-driven stepping (no double-stepping anomaly)
- Driven UKW (10 positions)
- Period: 10,000
- Power: 4-position selector (+ external/Sammler)
- Offered to Spain November 1931 (not purchased)

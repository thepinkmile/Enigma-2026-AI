## Enigma A

### Overview

The Enigma A (model Ch.A or model "A" per Scherbius 1923 catalogue) was the **first commercially marketed Enigma machine with a glow-lamp output**. Scherbius's very first ciphering machine used a
typewriter-style print wheel for output — the glow-lamp arrangement debuted with Enigma A. Only approximately 10 Enigma A units were manufactured.

## Enigma B

### Overview

The Enigma B (model Ch.B, internal code B-28 in Sweden) was a 3-rotor commercial Enigma with a 28-letter alphabet, sold primarily to businesses and the Swedish STAB (Ivar Kreuger's financial empire).
The Enigma B used 28-contact rotors (26 letters + 2 special characters) — making it distinct from the later 26-contact standard commercial Enigma C and D. The Swedish Army obtained a modified variant
known as the **A-133** (B-28 with Swedish alphabet modifications); Swedish State owned machine A-133 (inventory serial 133, hence "A-133").

### Technical Specifications

| Parameter | Value |
| ----------- | ------- |
| Official designation | Enigma B (model Ch.B) |
| Swedish designation | B-28; A-133 |
| Year introduced | 1925 |
| Rotor slots | 3 |
| Contacts per rotor | 28 (A-133 variant: 28 Scandinavian-alphabet variant) |
| Reflector | Settable UKW |
| Plugboard | None |
| ETW | 28-contact variant |
| Keyboard | QWERTZ + 2 additional |
| Units produced | ~30 estimated |
| Notable users | Swedish financial sector (STAB/Ivar Kreuger); Swedish Army (A-133) |

### Swedish A-133 Variant

The Swedish Army's A-133 used a modified 28-contact layout to accommodate Scandinavian characters (Å, Ä, Ö in place of the standard 2 null characters). It is the only known example of a national
alphabet-modified Enigma.

**Surviving example:** The Crypto Museum holds a Swedish A-133 obtained from the Swedish Army.

# Printing Enigmas (Handelsmaschine, PE26, H29/Enigma II)

| Field | Value |
| ------- | ------- |
| Who | ChiMaAG (Chiffriermaschinen AG), Berlin; Arthur Scherbius (designer) |
| What | Enigma machines with typewriter-style printed output rather than glow-lamps; sold commercially and to diplomatic services; designed for continuous-text encryption without manual transcription of lamp outputs |
| When | Handelsmaschine: 1923; PE26: 1926; H29/Enigma II: 1929–1930 |
| Where | Manufactured in Berlin, Germany (52.5200°N, 13.4050°E) |
| Related | [Arthur Scherbius](../profiles/arthur-scherbius.md), [Enigma D Commercial](enigma-d-commercial.md), [Enigma A and B](enigma-a-and-b.md) |

## Overview

The Printing Enigmas were a parallel line of Enigma development where the output was **printed on paper tape** rather than indicated by glow-lamps. This made them more suitable for commercial
telegraph and telex applications — operators did not need to manually copy lamp readings. Three main models are documented: the 1923 Handelsmaschine prototype, the 1926 PE26 ("Printing Enigma 26"),
and the 1929 H29/Enigma II series which was marketed more extensively.

## Handelsmaschine (1923)

| Parameter | Value |
| ----------- | ------- |
| Year | 1923 |
| Output | Typewriter-print mechanism (Schreibwerk) |
| Rotors | 3 cipher rotors |
| Contacts | 26 |
| Keyboard | QWERTZ |
| Units | Unknown (prototype series) |
| Status | Concept phase / very limited production |

The 1923 Handelsmaschine (literally "commercial machine") was Scherbius's first attempt at a printing variant — designed to make Enigma practical for commercial telegraph firms. It was not
commercially successful.

## H29/Enigma II (1929–1930)

| Parameter | Value |
| ----------- | ------- |
| Official designation | H29 / Enigma II |
| Year | 1929 |
| Output | Dual typewriter mechanism — both input and output typed simultaneously |
| Rotors | 3 cipher rotors |
| Contacts | 26 |
| ETW | QWERTZUIOASDFGHJKPYXCVBNML |
| Keyboard | QWERTZ (full typewriter with shift key) |
| Plugboard | None |
| Weight | ~50 kg (built into a full typewriter body) |
| Dimensions | Very large — comparable to a professional typewriter of the era |
| Units | ~2–3 known examples |
| Notable users | German Foreign Office (attempted but declined); Swiss Government (evaluated) |

The H29/Enigma II was the final commercial Enigma model marketed by ChiMaAG before the company's financial difficulties led to restructuring. It was a significant technological achievement but too
expensive and too large for widespread adoption.

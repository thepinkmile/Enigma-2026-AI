# Enigma KD (Mil Amt Enigma)

| Field | Value |
| ------- | ------- |
| Who | Heimsoeth und Rinke (H&R) designer; Militärisches Amt (Mil Amt) — successor to Abwehr within RSHA — operator |
| What | Enigma K variant with field-rewirable UKW-D and 9-notch rotors; used on Berlin–Madrid–Lisbon intelligence link |
| When | First appeared 3 December 1944; used until end of WWII |
| Where | Berlin, Germany (52.5200°N, 13.4050°E) → Madrid, Spain (40.4168°N, 3.7038°W) → Lisbon, Portugal (38.7169°N, 9.1399°W) |

## Overview

The Enigma KD was a late-war variant of the Enigma K used by the Militärisches Amt (Mil Amt), which was the successor intelligence organisation to the Abwehr after its dissolution in 1944. The KD's
defining features are its 9-notch rotors (providing very frequent and irregular stepping) and the **UKW-D field-rewirable reflector**, with wiring changed approximately every 3 weeks. It was used on
the Berlin–Madrid–Lisbon diplomatic/intelligence link. Whether it was ever broken by the Allies is unknown.

## Technical Specifications

| Parameter | Value |
| ----------- | ------- |
| Official designation | Enigma KD |
| Year introduced | First appeared 3 December 1944 |
| Rotors | 3 from a set of 6; each with **9 turnover notches** |
| Reflector | UKW-D (field-rewirable); wiring changed approximately every 3 weeks |
| Plugboard | None |
| ETW | QWERTZUIOASDFGHJKPYXCVBNML |
| Key change schedule | Rotor order + Ringstellung changed daily; UKW-D wiring changed every ~3 weeks |
| Units produced | Unknown (very small) |
| Broken by Allies | Unknown — traffic persisted to end of war |

## Surviving Examples

One machine found in 2009 in **FRA (Försvarets Radioanstalt / Swedish Intelligence)** archives — the only confirmed surviving Enigma KD.

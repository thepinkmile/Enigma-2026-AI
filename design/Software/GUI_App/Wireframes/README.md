# Enigma-NG — GUI Application Wireframes

This directory contains draw\.io wireframe and workflow diagrams for the Enigma-NG Avalonia desktop application\.
All wireframes target a **1920 × 1080 landscape** canvas using **Material Design 3 \(MD3\)** component conventions
and the **"Digital Museum" theme** \(Enigma Green \#004D40, Slate Grey, Amber \#FFB300 accents\)\.

## Files

| File | Description |
|---|---|
| `00-app-navigation-flow.drawio` | Full application navigation flowchart — screens, decisions, modal flows, and nav rail transitions |
| `01-main-dashboard.drawio` | Hardware Dashboard wireframe — Virtual Enigma panel, live telemetry cards, power source indicators, status bar |
| `02-rotor-configurator.drawio` | Rotor Configurator wireframe — 30-slot scrollable list, detail pane with wiring selector, ring/position settings |
| `03-daily-key-codebook.drawio` | Daily Key / Codebook Manager wireframe — config summary card, import drag-drop zone, JSON/PDF export, history list |
| `04-educational-timeline.drawio` | Educational Timeline wireframe — scrollable timeline with era bands, alternating event cards, filter bar, detail pane |
| `05-educational-map.drawio` | Educational Map wireframe — OpenStreetMap tile placeholder, location pins, sidebar list, expandable bottom sheet |
| `06-educational-profiles.drawio` | Historical Profiles wireframe — alphabetical filter sidebar, 3-column profile card grid, sliding detail pane |
| `07-educational-configurations.drawio` | Cipher Machine Configurations wireframe — accordion list with column headers, status chips, filter/sort bar, spec detail pane |

## Component Conventions

| Component | Style |
|---|---|
| Navigation Rail | 80px wide, Enigma Green \(\#004D40\) background; active item Amber \(\#FFB300\) |
| MD3 Card | Rounded rectangle, \#FAFAFA fill, \#CCCCCC border |
| Primary Button | \#004D40 fill, white label, rounded |
| FAB | Amber \(\#FFB300\) fill, rounded |
| Status OK chip | \#d5e8d4 / \#82b366 |
| Status Warning chip | \#fff2cc / \#d6b656 |
| Status Error/Broken chip | \#f8cecc / \#b85450 |

## Viewing

Open any `\.drawio` file in [draw\.io Desktop](https://github.com/jgraph/drawio-desktop/releases) or at [app\.diagrams\.net](https://app.diagrams.net)\.

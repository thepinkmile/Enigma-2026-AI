# Character Usage Rules

All non-ASCII characters used in design documents (any file outside `.copilot/`) must appear in the
approved table in `.copilot/allowed-character-matrix.md`.

Before using any non-ASCII character not already in the table, stop and ask the user for explicit
approval.

`.copilot/` files are not strictly bound by this rule, but new agent-authored content there should
follow it as best practice.
